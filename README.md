# StyleShop - E-commerce Website

A modern e-commerce website built with Next.js 15 and Tailwind CSS, featuring a responsive design and a beautiful user interface.

## Features

- Responsive design that works on all devices
- Product catalog with categories for men and women
- Featured products section
- Detailed product pages with size and color selection
- Modern and clean user interface
- Fast page loads with Next.js
- Beautiful styling with Tailwind CSS

## Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/styleshop.git
```

2. Navigate to the project directory:
```bash
cd styleshop
```

3. Install dependencies:
```bash
npm install
# or
yarn install
```

4. Start the development server:
```bash
npm run dev
# or
yarn dev
```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
